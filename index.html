What is canopy?

canopy is a web testing framework with one goal in mind: keep testing simple. canopy is built on top of selenium and written in f#. It's syntax is clean and concise. As an added benefit you can easily leverage your existing .Net code to help you test.

Getting started

  1. Create new F# console application
  2. Open NuGet console
  3. Type: 
    PM> Install-Package canopy
  4. Type in Program.fs: 
    open canopy
start "firefox"
url "http://google.com"
"input[type='text']" << "hello world"
  5. Run!

Examples

Below are examples of all the basic commands in the framework.

Starting a Browser

start firefox
start ie
start chrome

Describe the Test

describe "some functionality should work"

Browse to a Url

url "http://lefthandedgoat.github.com/canopy/"

Assert on a Url

on "http://lefthandedgoat.github.com/canopy/"

Clicking an Element

Note: Identify the element you want to interact with by using CSS Selectors 

click "#submit"

Setting a TextBox

"#firstName" << "John"

Clear a TextBox

clear "#lastName"

Assert the Value of an Element

"#firstName" == "John"

Assert the Value of an Element Not Equal

"#firstName" != "Johnny"

Assert One of Many Elements Has a Value

".items" *= "Item 1"

Assert No Elements Have Value

".items" !*= "Item 100"

Assert Element Selected

selected "#radio1"

Assert Element Not Selected

deselected "#checkbox"

reload ()
